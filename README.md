# Sign Language Detection

A real-time Sign Language Detection system developed using Python, OpenCV, MediaPipe, and Machine Learning techniques. The project detects and recognizes hand gestures and converts sign language into text output.

## Features

* Real-time hand gesture detection
* Sign recognition using webcam
* MediaPipe hand landmark tracking
* Text output generation
* User-friendly interface

## Technologies Used

* Python
* OpenCV
* MediaPipe
* TensorFlow/Keras

## Applications

* Communication support for deaf and mute individuals
* Gesture-based interaction systems
* Educational learning tool

## Future Enhancements

* Speech output integration
* Sentence prediction
* Multi-language sign recognition
