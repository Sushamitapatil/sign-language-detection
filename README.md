# 🤟 Sign Language Detection System

A real-time **Sign Language Detection System** developed using **Python, OpenCV, MediaPipe, and Deep Learning** techniques. This project recognizes hand gestures from American Sign Language (ASL) and converts them into text and speech output to support communication for deaf and hard-of-hearing individuals.

---

## 📌 Features

* Real-time hand gesture recognition using webcam
* ASL alphabet and gesture detection
* MediaPipe hand landmark tracking
* Text generation from detected gestures
* Text-to-Speech (TTS) output support
* User-friendly graphical interface
* Real-time prediction and feedback
* CNN-based gesture classification for improved accuracy

---

## 🎯 Objectives

* Recognize ASL hand gestures and convert them into readable text

## 🏗️ System Architecture

### 1. Input Module

Captures real-time hand gestures using a webcam or camera.

### 2. Preprocessing Module

* Hand detection using MediaPipe
* Image normalization and frame processing

### 3. Feature Extraction Module

* Detects hand landmarks and keypoints
* Tracks finger and hand movement patterns

### 4. Sign Classification Module

* Uses Convolutional Neural Networks (CNN) for gesture recognition

### 5. Output Module

* Converts gestures into text
* Generates speech output using TTS
* Displays results through GUI
